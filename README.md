# anmar-ui

Design system compartilhado dos apps Anmar (`anmar-apps`, `anmar-rebanho`, `anmar-leite`, ...).

> Repo privado. Acesso restrito via SSH key (deploy key em CI).

## Install

```bash
pip install git+ssh://git@github.com/anderson-pids/anmar-ui.git@v0.1.0
```

## Use

```python
from anmar_ui import mount as mount_ui
mount_ui(app, templates)
```

Veja spec completo em `anmar-apps` repo: `docs/superpowers/specs/2026-05-07-anmar-ui-design-system-design.md`.
